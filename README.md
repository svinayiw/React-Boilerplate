# First Ghar Adviser Frontend

> ReactJS, TypeScript, REST 

## Installation Guide
```
# clone repository
$ git clone git@github.com:svinayiw/React-Boilerplate.git

# install dependencies
$ yarn

# copy and modify the .env.example to .env

$ cd React-Boilerplate

# start 
$ yarn start

```

## Prettier

```
# Prettier Check 
$ yarn prettier:check

# Prettier Fix 
$ yarn prettier:fix 
```

## Create Build

```
$yarn build
```
